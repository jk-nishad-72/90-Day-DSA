## 1Ô∏è‚É£ 804. Unique Morse Code Words

### üîπ Problem Statement
Given an array of strings `words`, convert each word into its **Morse code representation** and return the number of **unique transformations**.

Each lowercase English letter has a fixed Morse code mapping.

---

### üîπ Approach

1. Create a **map** that stores each letter (`a` to `z`) with its corresponding Morse code.
2. Use a **Set** to store unique Morse transformations.
3. For each word:
   - Traverse each character
   - Convert it to Morse using the map
   - Concatenate the Morse codes to form one string
4. Add the transformed string to the Set.
5. Return the size of the Set.

---

### üîπ Algorithm

- Initialize a Morse code map
- Initialize an empty Set
- Loop through each word:
  - Convert the word into Morse
  - Store it in the Set
- Return `Set.size`

---

### üîπ JavaScript Code

```js
var uniqueMorseRepresentations = function(words) {

    let map = {
        a: ".-",
        b: "-...",
        c: "-.-.",
        d: "-..",
        e: ".",
        f: "..-.",
        g: "--.",
        h: "....",
        i: "..",
        j: ".---",
        k: "-.-",
        l: ".-..",
        m: "--",
        n: "-.",
        o: "---",
        p: ".--.",
        q: "--.-",
        r: ".-.",
        s: "...",
        t: "-",
        u: "..-",
        v: "...-",
        w: ".--",
        x: "-..-",
        y: "-.--",
        z: "--.."
    };

    let set = new Set();

    for (let i = 0; i < words.length; i++) {
        let word = words[i];
        let ans = '';

        for (let j = 0; j < word.length; j++) {
            ans += map[word.charAt(j)];
        }

        set.add(ans);
    }

    return set.size;
};
````

---

### üîπ Time & Space Complexity

* **Time Complexity:** `O(n * m)`
  (`n` = number of words, `m` = average word length)
* **Space Complexity:** `O(n)` for storing transformations

## 2Ô∏è‚É£ 2062. Count Vowel Substrings of a String

### üîπ Problem Statement

Given a string `word`, count all substrings that:

1. Contain **only vowels**
2. Contain **all five vowels** (`a, e, i, o, u`) at least once

---

### üîπ Approach

1. Use a **Set** to store vowels.
2. For each index `i`, start forming substrings.
3. Expand the substring until:

   * A consonant is found ‚Üí stop
4. Track distinct vowels using another Set.
5. Whenever the Set size becomes `5`, increment the count.

---

### üîπ Algorithm

* Loop over each starting index
* Create a new Set for vowels seen
* Extend substring character by character
* Break if a consonant appears
* Count valid substrings when all 5 vowels are present

---

### üîπ JavaScript Code

```js
var countVowelSubstrings = function(word) {

    const vowels = new Set(['a', 'e', 'i', 'o', 'u']);
    let count = 0;

    for (let i = 0; i < word.length; i++) {
        let seen = new Set();

        for (let j = i; j < word.length; j++) {
            if (!vowels.has(word[j])) break;

            seen.add(word[j]);

            if (seen.size === 5) {
                count++;
            }
        }
    }

    return count;
};
```

---

### üîπ Time & Space Complexity

* **Time Complexity:** `O(n¬≤)`
* **Space Complexity:** `O(1)` (only 5 vowels max)

## ‚úÖ Key Takeaways

* Use **Set** to handle uniqueness efficiently
* Break early when conditions fail (optimization)
* Nested loops are acceptable when constraints are small
* These problems strengthen **string traversal + set logic**
